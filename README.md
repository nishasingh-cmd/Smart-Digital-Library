# 🚀 BookBloom Genie — AI-Powered Digital Library Platform

BookBloom Genie is a modern, full-stack digital library platform built using **React (Vite)** and **Flask**, designed to deliver a premium, personalized reading and audiobook experience.

It combines elegant UI/UX with scalable backend architecture to simulate a real-world startup-grade product.

---

## 🌟 Overview

BookBloom Genie transforms the traditional library experience into a **smart, interactive, and user-centric platform**. Users can explore books, manage personal collections, listen to audiobooks, and receive intelligent recommendations — all within a beautifully designed interface.

---

## 🧠 Core Features

### 🔐 Authentication (Simulated)

* Secure login/logout flow using localStorage
* Protected routes for authenticated users

### 📚 Smart Book Discovery

* Dynamic book listings from backend APIs
* Real-time search and filtering
* Category-based exploration

### ❤️ Favorites & Personal Library

* Add/remove favorites
* Persistent user preferences
* Unified “My Library” dashboard

### 🎧 Audiobook Experience

* Custom audio player UI
* Playback controls (play/pause/seek)
* Interactive listening interface

### 🤖 AI-Powered Recommendations

* Personalized book suggestions
* Trending & recommended sections
* Scalable logic for future ML integration

### 📄 Book Detail Pages

* Detailed descriptions
* Reviews & ratings
* Related content suggestions
